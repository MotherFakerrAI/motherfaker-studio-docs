## 📚 文档结构

```
motherfaker-studio-docs/
├── README.md                    # 文档中心首页（你在这里）
├── products/                    # 【对外】产品介绍
│   ├── README.md               # 产品总览
│   └── {product-name}/
│       └── index.md            # 产品主文档
├── projects/                    # 【对内】项目过程
│   ├── README.md
│   └── {project-name}/
│       ├── dev-log.md          # 开发日志
│       └── decisions.md        # 技术决策

│   ├── workflow.md             # 工作流程
│   └── agent-protocol.md       # Agent 协议
└── assets/                      # 公共资源
    ├── logo/
    └── templates/              # 文档模板
```
