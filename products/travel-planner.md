# 🧳 智能旅行规划助手

## 项目概述

| 项目 | 内容 |
|------|------|
| 名称 | 智能旅行规划助手 |
| 描述 | AI 旅行规划聊天界面，动态生成行程卡片 |
| 技术栈 | Next.js 14 + TypeScript + CopilotKit + Tailwind CSS |
| 仓库 | https://github.com/MotherFakerrAI/travel-planner |
| 创建时间 | 2026-03-06 |

---

## 功能清单

- ✅ AI 对话式旅行规划
- ✅ 动态生成行程卡片
- ✅ 智能推荐目的地和路线
- ✅ 支持多轮对话调整行程

---

## 技术实现

- Next.js 14 框架
- TypeScript 类型安全
- CopilotKit AI 对话能力
- Tailwind CSS 样式
