# 🔍 代码审查助手

## 项目概述

| 项目 | 内容 |
|------|------|
| 名称 | 代码审查助手 |
| 描述 | 上传代码片段，AI 审查并生成可视化报告，支持交互式修改 |
| 技术栈 | Next.js 14 + TypeScript + CopilotKit + Groq（免费 LLM） |
| 仓库 | https://github.com/MotherFakerrAI/code-reviewer |
| 创建时间 | 2026-03-06 |

---

## 功能清单

- ✅ 上传代码片段
- ✅ AI 自动审查代码
- ✅ 生成可视化审查报告
- ✅ 支持交互式修改建议
- ✅ 使用 Groq 免费 LLM

---

## 技术实现

- Next.js 14 框架
- TypeScript 类型安全
- CopilotKit AI 对话能力
- Groq 免费 LLM 服务
