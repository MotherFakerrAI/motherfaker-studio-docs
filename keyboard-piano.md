# 🎹 keyboard-piano 项目文档

## 项目概述

| 项目 | 内容 |
|------|------|
| 名称 | keyboard-piano（键盘钢琴） |
| 描述 | 键盘钢琴 - 按键演奏 + 粒子特效 |
| 技术栈 | HTML（单文件） |
| 仓库 | https://github.com/MotherFakerrAI/keyboard-piano |
| 预览 | https://motherfakerrai.github.io/keyboard-piano/ |
| 创建时间 | 2026-03-03 |

---

## 功能清单

- ✅ 键盘按键（A-L）发出不同音符
- ✅ 鼠标点击琴键也能弹奏
- ✅ 按下时屏幕出现粒子炸开效果
- ✅ 粒子之间有连线跟随舞动
- ✅ 不同音符对应不同颜色

---

## 技术实现

- 单 HTML 文件 + CDN（无外部依赖）
- Web Audio API 合成音符
- Canvas 绘制粒子动画
- 响应式设计
